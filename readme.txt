Este é um plugin para você mostrar como você se sente hoje. Triste, feliz, alegre, cansado e muitas outras opções!

== Description ==
Este plugin coloca uma carinha (emoticon) e um texto mostrando como você se sente hoje. Possui mais de 30 opções de sentimentos e na próxima versão do plugin, você poderá criar os seus próprios status.
